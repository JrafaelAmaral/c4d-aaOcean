# aaOceanC4D

aaOcean Shader and Deformer by Amaan Akram for Cinema 4D, ported by
Niklas Rosenstein.

## Build Instructions

  [Craftr]: https://craftr.net

* Clone the repository in the Cinema 4D application plugins directory
* Update git submodules
* Install [Craftr][]
* Run `craftr export`
* Run `craftr build`
